# Next.js + Strapi Todo App

A full-stack Todo application built as part of a startup assignment using Next.js and Strapi.

## Features

- User Authentication (Signup/Login)
- JWT-based Authentication
- Protected Routes
- Create Todo
- Read User-specific Todos
- Update Todo Status
- Delete Todo
- Logout Functionality

## Tech Stack

### Frontend
- Next.js 14
- Tailwind CSS
- Axios

### Backend
- Strapi v5
- SQLite
- Users & Permissions Plugin

## What I Explored

- Strapi CMS
- Content Types & Relations
- JWT Authentication
- Route Protection
- CRUD APIs
- Frontend + Backend Integration
